### 1. Verify UI Elements on Load
**Test Case**: Verify all necessary fields and buttons are visible  
- **Precondition**: User is authenticated and on the Coupons Page.  
- **Steps**:  
  1. Navigate to the CouponsPage.  
  2. Ensure the following UI elements are visible:  
     - "Coupon Management" title.  
     - "Search coupons" input field with a search icon.
